import { Tabs, TabItem } from '@astrojs/starlight/components';

Namespace: `ARCtrl`

`XlsxController` bundles helpers to convert ARC entities to/from `FsWorkbook` and XLSX files.

**Platform note:** JS/TS targets expose the `*Async` methods; .NET/Python targets expose the sync `fromXlsxFile`/`toXlsxFile` methods.

## Fields
None.
