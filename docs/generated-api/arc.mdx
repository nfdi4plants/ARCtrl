### Constructor
**Summary:** Create an ARC with investigation metadata, ISA objects, optional file system, and license.
**Parameters:** `identifier: string`, optional metadata (`title`, `description`, `submissionDate`, `publicReleaseDate`), collections (`ontologySourceReferences`, `publications`, `contacts`, `assays`, `studies`, `workflows`, `runs`, `registeredStudyIdentifiers`, `comments`, `remarks`), `fs: FileSystem.FileSystem`, `license: License`
**ReturnType:** [ARC](./arc)
<Tabs>
<TabItem label="FSharp" icon='seti:f-sharp'>
```fsharp
open ARCtrl
let arc = ARC(
    identifier = "ARC1",
    title = "My ARC",
    assays = ResizeArray(),
    fs = FileSystem.FileSystem.create(FileSystem.FileSystemTree.createRootFolder [||])
)
```
</TabItem>
<TabItem label="JavaScript" icon='seti:javascript'>
```ts
import { ARC, FileSystem, FileSystemTree } from "@nfdi4plants/arctrl";

const arc = new ARC(
  "ARC1",
  "My ARC",
  undefined,
  undefined,
  undefined,
  undefined,
  undefined,
  undefined,
  [],
  [],
  [],
  [],
  [],
  [],
  FileSystem.create(FileSystemTree.createRootFolder([])),
);
```
</TabItem>
<TabItem label="Python" icon='seti:python'>
```py
from arctrl import ARC, FileSystem, FileSystemTree

arc = ARC(
    "ARC1",
    title="My ARC",
    assays=[],
    fs=FileSystem.create(FileSystemTree.createRootFolder([])),
)
```
</TabItem>
</Tabs>
