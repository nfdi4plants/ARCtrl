import { Tabs, TabItem } from '@astrojs/starlight/components';

Namespace: `ARCtrl`

## Fields
- `Identifier`: string (unique in investigation)
- `Investigation`: [ArcInvestigation](./arc-investigation) option (internal)
- `Title` / `Description`: string option
- `MeasurementType` / `TechnologyType` / `TechnologyPlatform`: [OntologyAnnotation](./ontology-annotation) option
- `WorkflowIdentifiers`: string array
- `Tables`: [ArcTable](./arc-table) array
- `Datamap`: [Datamap](./datamap) option
- `Performers`: [Person](./person) array
- `CWLDescription`: `CWLProcessingUnit` option
- `CWLInput`: `CWLParameterReference` array
- `Comments`: [Comment](./comment) array
- `StaticHash`: int
- `FileName`: run ISA filename
- `WorkflowIdentifierCount` / `WorkflowCount`
- `Workflows`: resolve identifiers to workflows in investigation
- `VacantWorkflowIdentifiers`: identifiers without backing workflow objects
