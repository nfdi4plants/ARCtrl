Namespace: `ARCtrl`

## Fields
- `Identifier`: string (unique in investigation)
- `Investigation`: [ArcInvestigation](./arc-investigation) option (set internally)
- `Title`: string option
- `Description`: string option
- `SubmissionDate`: string option
- `PublicReleaseDate`: string option
- `Publications`: [Publication](./publication) ResizeArray
- `Contacts`: [Person](./person) ResizeArray
- `StudyDesignDescriptors`: [OntologyAnnotation](./ontology-annotation) ResizeArray
- `Tables`: [ArcTable](./arc-table) ResizeArray
- `Datamap`: [Datamap](./datamap) option
- `RegisteredAssayIdentifiers`: string array
- `Comments`: [Comment](./comment) ResizeArray
- `StaticHash`: int
- `FileName`: study ISA filename
- `RegisteredAssayIdentifierCount` / `RegisteredAssayCount`
- `RegisteredAssays`: resolves registered identifiers to assays in parent investigation
- `VacantAssayIdentifiers`: identifiers without backing assay objects
- `FileName`: ISA study filename

## Methods

### Constructor
**Summary:** Create a study with identifiers, optional metadata, publications, contacts, design descriptors, tables, datamap, registered assay ids, and comments.
**Parameters:**
- `identifier`: string
- `title`: string option
- `description`: string option
- `submissionDate`: string option
- `publicReleaseDate`: string option
- `publications`: [Publication](./publication) array option
- `contacts`: [Person](./person) array option
- `studyDesignDescriptors`: [OntologyAnnotation](./ontology-annotation) array option
- `tables`: [ArcTable](./arc-table) array option
- `datamap`: [Datamap](./datamap) option
- `registeredAssayIdentifiers`: string array option
- `comments`: [Comment](./comment) array option
**ReturnType:** [ArcStudy](./arc-study)
<Tabs>
<TabItem label="FSharp" icon='seti:f-sharp'>
```fsharp
open ARCtrl
let study = ArcStudy(
  identifier = "Study1",
  submissionDate = "2024-01-01",
  publications = ResizeArray(),
  contacts = ResizeArray(),
  studyDesignDescriptors = ResizeArray(),
  tables = ResizeArray()
)
```
</TabItem>
<TabItem label="JavaScript" icon='seti:javascript'>
```ts

```
</TabItem>
<TabItem label="Python" icon='seti:python'>
```py

```
</TabItem>
</Tabs>
