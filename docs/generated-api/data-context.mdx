## Static Methods

### fromData
**Summary:** Promote a `Data` into a `DataContext` with optional ontology/metadata fields.
**Parameters:** `data: [Data](./data)`, `explication: [OntologyAnnotation](./ontology-annotation) option`, `unit: [OntologyAnnotation](./ontology-annotation) option`, `objectType: [OntologyAnnotation](./ontology-annotation) option`, `label: string option`, `description: string option`, `generatedBy: string option`
**ReturnType:** [DataContext](./data-context)
<Tabs>
<TabItem label="FSharp" icon='seti:f-sharp'>
```fsharp
let dc = DataContext.fromData(data, explication = oa)
```
</TabItem>
<TabItem label="JavaScript" icon='seti:javascript'>
```ts
const dc = DataContext.fromData(data, oa);
```
</TabItem>
<TabItem label="Python" icon='seti:python'>
```py
dc = DataContext.fromData(data, explication=oa)
```
</TabItem>
</Tabs>

### createAsPV
**Summary:** Build a property-value style `DataContext` with optional ontology/unit.
**Parameters:** `alternateName: string option`, `measurementMethod: [OntologyAnnotation](./ontology-annotation) option`, `description: string option`, `category: [OntologyAnnotation](./ontology-annotation) option`, `value: string option`, `unit: [OntologyAnnotation](./ontology-annotation) option`
**ReturnType:** [DataContext](./data-context)
<Tabs>
<TabItem label="FSharp" icon='seti:f-sharp'>
